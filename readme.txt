== Description ==
Coronavirus Cases Tracker Plugin is used to display Novel Coronavirus (COVID-19) outbreak live Updates in WordPress page/posts/widgets section.
With the help of shortcode,You can represent total Infected,Deaths,Recovered live data in the WordPress.
Since version 1.1, you can also create country based COVID-19 outbreak data card.

<h4>Use this shortcode in your page/posts/widget section</h4>
<h4>Shortcodes with Default Settings</h4>
<pre>[cvct]</pre>
<h4>Shortcodes with Advance Settings</h4>
<pre>[cvct title="COVID-19 Global  Stats" label-total="Total Cases" label-deaths="Death Cases" label-recovered="Recovered Cases"  bg-color="#ddd" font-color="#000"]</pre>

<h4>Country Based COVID-19 outbreak stats shortcodes</h4>
[cvct title="Coronavirus Stats"  country-code="US" label-total="Total Cases" label-deaths="Death Cases" label-recovered="Recovered Cases"  bg-color="#23282D" font-color="#fff"]
<br/>
[cvct title="Corona Virus Stats"  country-code="IT" label-total="Total Cases" label-deaths="Death Cases" label-recovered="Recovered Cases"  bg-color="#CB3A26" font-color="#fff"]
<h5>Here is the shortcode settings details</h5>
<ul>
<li>country-code = <strong>all/particular country eg:US,IN etc.</strong> (By default it show the all country data.)</li>
<li>title = <strong>Any text here...</strong> (You Can Change the title name and by default it take Global Stats.)</li>
<li>label-total = <strong>Any text here...</strong> (By Default, it set Total cases text)</li>
<li>label-deaths = <strong>Any text here...</strong> (By Default it set Deaths text.)</li>
<li>label-recovered = <strong>Any text here...</strong> (By Default it set New Recovered text.)</li>
<li> bg-color = <strong>Any Color here...</strong> (Using this  attribute You Can change background color.)</li>
<li>font-color = <strong>Any Color here...</strong> (Using this  attribute You Can change font color.)</li>
</ul>
